## Notes

This is a list of some of the most useful Chrome Browser URLs for its internal web pages along with a brief description for each one. 



Note: You can display the list of most Chrome URLs by typing chrome://about/ in the browser. 



Chrome URL

Description

chrome://about

List of most internal Chrome URLs

chrome://accessibility

Displays accessibility information for each tab open in the browser, and whether the feature is turned on globally.

chrome://app-service-internals

List of all apps still installed in Chrome along with permissions and settings

chrome://apps

Lists all installed apps

chrome://bluetooth-internals/

Displays information about connected Bluetooth adapters and devices

chrome://bookmarks

Shortcut to the Bookmark Manager

chrome://chrome-urls

Same as chrome://about

chrome://components/

Lists all internal components with the option to check for updates for each component individually

chrome://crashes/

Lists all recently logged and reported crashes with the option to clear the logs as well.

chrome://device-log/

Lists all connected devices that Chrome can work with such as mouse, printer, etc.

chrome://dino

Loads the dino game (full screen)

chrome://discards/

List of all tabs and the option to discard individual tabs

chrome://downloads

Shortcut to the Downloads Manager

chrome://download-internals

Displays the status of current downloads with the option to start a new download.

chrome://extensions

Shortcut to Extensions Manager

chrome://flags

Lists experimental features that may or may not be integrated into the browser in the future

chrome://gpu

Lists detailed information on the GPU and supported features (e.g., hardware acceleration)

chrome://help

Shortcut to the About page

chrome://history

Shortcut to Browser History

chrome://inspect

Provides access to DevTools unless disabled by Admin

chrome://management
Displays whether the profile you are signed into is managed
chrome://media-internals

Displays information on recently played media

chrome://newtab

Shortcut to a New Tab page (i.e., new-tab-page)

chrome://ntp-tiles-internals

Displays configuration details for New Tab pages

chrome://omnibox

Display address bar input results on this page, includes search, shortcuts and history information

chrome://policy

Lists all applied policies and an option to export them

chrome://predictors

Lists all auto complete and resource prefetch predictors based on user history

chrome://print

Shortcut to Print Preview page

chrome://safe-browsing

Displays configuration details for Safe Browsing

chrome://settings

Shortcut to Chrome Settings

chrome://signin-internals

Displays information on sign-in status and other details for the current user account

chrome://site-engagement

Displays engagement scores for all currently open and recently visited sites

chrome://sync-internals

Displays detailed information about user account synchronization

chrome://system/

Provides general OS details regarding system, sync, memory usage, etc.

chrome://terms

Shortcut to Google Chrome's Terms of Service

chrome://user-actions/

Lists user's actions when opened in background

chrome://version

Displays general information including browser version, user agent, JavaScript, and WebKit versions, as well as Command Line parameters
